# Sistema ATM em C

Integrantes: Bruno Yudi Moritaka Kanashiro

Descrição: O sistema simula algumas das funções de um ATM em C.

Funcionalidades implementadas: as funcionalidades implementadas são a de exibir o menu, consultar o saldo, realizar o saque e realizar o deposito.

Instruções: Selecione a função desejada apertando o número de acordo com o menu e então digite o valor sacado ou depositado.
Instruções do saque: valores maiores que mil, menores ou iguais a zero e maiores que o saldo atual não serão aceitos.
instruções do deposito: valores menores ou iguais a zero não serão aceitos.
